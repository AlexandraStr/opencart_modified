Vger - Pre-set Admin Order Comments


Adds buttons to the order history screen to speed up adding standard order comments. Order comments can be modified in the order Comments module.

Thank you for downloading this extension. Please follow these instructions to proceed with the installation. To configure your new pages please see the instructions near the bottom of this page.

REMEMBER TO BACKUP YOUR FILES BEFORE MAKING ANY CHANGES

# General Information:

In this zip file you will find the following files and folders:

/upload: - If using vQmod this folder contains everything you need.

vqmod- In here you will find an xml vQmod file that will alter the necessary files to make the theme work.

admin folder - Contains the necessary files for the extension and should be uploaded 'as is' as no files are over written.

Did I mention, BACKUP YOUR FILES FIRST.

# Installation without using vQmod:

Upload the admin folder (in the /upload folder) and then follow the instructions below to manually alter existing files.

# Installation using vQmod:

Upload the contents of the upload folder

If you have problems with the extension it may be because the vqmod is failing or conflicting with other vqmods. Before taking any further action please check your vqmod.log file.  If you have a problem with it then you may need to adjust the xml file to get it to work properly. This is not something that I can do for you (it will most likely be due to a conflict with an existing vQmod or a previously altered file) so if you can't manage to get the vQmod working then you'll need to do the manual file edits.

# Accessing your new extension:

Navigate to admin > Extensions> Modules.  You will see a new module called 'Order Comments'

Fill in any or all off the order comments that you need.

Add a simple one or two word Title for each comment so you can identify it on the Order History page.

Navigate to an order and click on the Order History tab and you will see the new button(s) at the bottom.  Click on one to add comments to the comments box.
